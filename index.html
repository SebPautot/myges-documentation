MYGES API DOCUMENTATION

Authenticate a user and get a token

Method :

GET https://authentication.kordis.fr/oauth/authorize?response_type=token&client_id=skolae-app

Parameters :

There is no custom parameter

Headers :

Autorization : Basic {credentials}
*Convert a string with the value username:password to Base64 to get the appropriate value of {credentials}

Redirects to an address

The value query parameter access_token of the redirection address (comreseaugesskolae:/oauth2redirect#) is the token you need to use. In the documentation the token will be called {token}, you will need to replace {token} with your token value.

Use the token to get and modify user informations

Method :

GET https://api.kordis.fr/me/

Parameters :

year : the year the data is from

agenda

startTime and endTime are used to get an interval

startTime : the start time in milliseconds since the Unix Epoch

endTime : the end time in milliseconds since the Unix Epoch

news

page : a page number (int)

Headers :

Autorization : bearer {token}

Returns

They all return an array
